Working With Combos

1.	To activate the combos functionality, you need to have cuts defined.
2.	Click on the combos Icon (  ) to launch the combos functionality. A combos window will appear that lists the available cuts that can be combined. Each available cut is numbered on the display and in the Define Groups box
3.	Form a group by selecting cuts. To do this, you click on the box below ‘Group’ across from desired cut. The line turns blue to indicate that it has been selected.
4.	If you want to allow the cut to be ‘flipped’ then hold the control key down and click on the box below ‘Flip’. An ‘X’ will appear in the box indicating that the cut can be flipped as a possible choice.
5.	Items can be selected or deselected by clicking on them.
6.	Once a set of cuts has been put together, clicking on ‘Combine’ forms a group, which will then appear in the ‘Process Groups’ box below. 
7.	In the Process Groups box you click on the button below ‘Show Options’ to display the available combinations.
8.	Click on ‘Delete Group’ to delete the group.
